# Command Router

Use this guide to decide which kind of AI tool is best suited for a task.

## ChatGPT or Similar Chat Agents

Best for:

- Planning work
- Reviewing strategy
- Synthesizing notes
- Writing reports, briefs, and summaries
- Creating checklists and prompts

Use when the task is primarily about thinking, writing, review, or communication.

## Codex or Similar Coding Agents

Best for:

- Editing repository files
- Creating templates
- Updating Markdown structures
- Running validation commands
- Managing code or documentation changes

Use when the task requires file operations, repository hygiene, validation, or repeatable edits.

## Claude, Gemini, or Other Long-Context Reviewers

Best for:

- Reading long source packs
- Providing independent review
- Comparing multiple documents
- Checking clarity, completeness, or contradictions

Use these as optional reviewers when a second perspective or larger context window is useful.

## Human Review

Always involve a human for:

- Destructive edits
- Public-facing changes
- Legal, financial, medical, or compliance decisions
- Anything involving sensitive data
- Decisions that affect real people, money, or reputation

Keep routing practical and tool-agnostic. The best tool is the one that can perform the task clearly, safely, and reviewably.
