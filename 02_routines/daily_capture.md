# Daily Capture Routine

Use this checklist at the end of a work session.

## Checklist

- [ ] Capture important sources in `01_wiki/source_index.md`.
- [ ] Add or update related project rows in `01_wiki/project_index.md`.
- [ ] Save reusable prompts or workflows to `01_wiki/skill_index.md`.
- [ ] Record meaningful changes in `02_routines/maintenance_log.md`.
- [ ] Remove accidental private details before committing.
- [ ] Mark uncertain items as unverified.
