# Weekly Rewiki Routine

Use this routine once a week to keep the workspace useful.

## Review Checklist

- [ ] Review new source index entries.
- [ ] Remove duplicate or unclear project rows.
- [ ] Promote repeated workflows into reusable skills.
- [ ] Archive stale fictional examples if they no longer help.
- [ ] Check that maintenance logs explain why changes were made.
- [ ] Confirm no private data or internal paths were introduced.

## Output

At the end of the review, add a short entry to `02_routines/maintenance_log.md` with:

- Date
- Files reviewed
- Changes made
- Follow-up items
