# AGENTS.md

Operating rules for AI agents working inside this repository.

## Start Here

Before making changes, always read:

- `README.md`
- `00_system/update_rules.md`
- Relevant indexes in `01_wiki/`

## Core Rules

- Work only inside this repository.
- Do not invent facts, sources, dates, decisions, or metrics.
- Keep examples fictional unless the user explicitly provides public information.
- Never commit secrets, credentials, API keys, OAuth tokens, private data, or internal file paths.
- Do not modify source material destructively.
- Prefer additive updates: append, annotate, or create a new dated note.
- Explain risky changes before applying them.
- Keep the difference between sources, indexes, templates, and examples clear.

## Editing Guidance

- Make small, focused changes.
- Preserve existing structure unless asked to improve it.
- Add maintenance log entries for meaningful updates.
- Mark uncertain items clearly instead of presenting them as facts.
- Use concise professional language.

## Public Repository Safety

This is intended to be a public open-source repository. Assume every file may be read by anyone.
