# Philosophy

AI tools are most useful when they are surrounded by good working habits.

This starter kit treats AI work as an operating system made of:

- Clear context
- Traceable sources
- Explicit decisions
- Reusable prompts
- Review routines
- Maintenance logs

The repository is intentionally simple. Markdown files are easy to read, easy to edit, and easy to review. The structure is meant to help people build continuity without requiring a database, dashboard, or custom application.

## Principles

- Structure beats memory.
- Review beats blind automation.
- Small routines beat occasional cleanup.
- Fictional examples beat accidental disclosure.
- Human judgment remains responsible for final decisions.

## For Non-Engineers

You do not need to code to use this system. Treat the files as shared operating notes. The important habit is to keep sources, decisions, and AI instructions clear enough that another person, or another AI session, can understand what happened.
